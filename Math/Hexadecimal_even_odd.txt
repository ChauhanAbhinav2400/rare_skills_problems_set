======================================================================================
Quickly Recognizing if a Hexadecimal Number is Even or Odd
======================================================================================

Description
In binary, a number is even if its last bit is 0 and odd if its last bit is 1.

We can use the same rule for hexadecimal numbers.
Each hex digit is 4 bits, so we only need to check the least significant hex digit.

Process:

Convert the last hex digit into 4-bit binary.
Check the last bit:
If it is 0 → the number is even.
If it is 1 → the number is odd.
Example 1
0xA (last digit A = 1010)
Last bit = 0 → even

Example 2
0xF (last digit F = 1111)
Last bit = 1 → odd

Example 3
0x3B (last digit B = 1011)
Last bit = 1 → odd

Example 4
0x64 (last digit 4 = 0100)
Last bit = 0 → even
======================================================================================
